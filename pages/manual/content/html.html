HTML

Básicos

A Linguagem de Marcação de Hiper-Texto é a tecnologia principal mais utilizada em webdev. É a base de todo o desenvolvimento das páginas online e offline, e não tão complicada quando são compreendidos os básicos absolutos.

HTML funciona sobre um sistema de tags, representadas por uma sintaxe específica. Um exemplo de HTML: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Cada tag em HTMl tem uma função diferente:

  • html: define os confins do documento HTML.
  • head: usada para definir bloco onde são inseridos os metadados da página.
  • title: define o título da página.
  • meta: usada para definir metadados individuais como o ícone, palavras-chave, descrição da página, dentre outros.
  • body: onde é colocado o conteúdo principal a ser exibido na página.

Como HTML é uma tecnologia sempre em desenvolvimento, tags entram em desuso constantemente, tal como tags novas são criadas para extender a funcionalidade da linguagem. Por exemplo, as tags center e font não são mais suportadas em HTML5, a versão mais recente de HTMl.
