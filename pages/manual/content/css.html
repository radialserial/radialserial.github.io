CSS

Básicos

A linguagem de Folhas de Estilo em Cascata é um sistema de introduzir estilos avançados em HTML de forma hierárquica. Um exemplo: 

footer {
    position: fixed;
    left: 0;
    bottom: 0;
    width: 100%;
    text-align: center;
    background-color: #282A36;
}

Esse bloco controla o estilo da tag footer, que age como o rodapé da página. Cada um dos atributos controla uma característica diferente do rodapé; width controla a largura, background-color controla a cor do fundo, e assim por diante.
