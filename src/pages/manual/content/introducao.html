Introdução

World Wide Web

  Originalmente envisionada por Tim Berners-Lee enquanto trabalhava na CERN nos anos 80, a rede de computadores mundial é um sistema de transmissão e gerência de documentos de texto, representados primáriamente por três tecnologias: HTML, CSS, e JavaScript.

Conforme a web evoluiu, porém, seu escopo transcendeu seu objetivo inicial, e se tornou algo completamente diferente. É difícil observar um website hoje em dia e interpretálo como um mero documento de texto.

De simples manuais em texto comum a vídeos, programas, e até mesmo videogames, as páginas da web contemporânea são capazes das mais diversas utilidades, dependendo da necessidade do usuário.

Funcionamento

A web, resumidamente, funciona em um sistema de clientes e servidores, onde clientes acessam servidores para receber páginas codificadas que são entçao interpretadas pelo navegador utilizado.

Tais páginas tem sua estrutura divida em três partes: conteúdo, estilo, e funcionalidade - cada uma dessas partes é representada por uma das linguages utilizadas em desenvolvimento web.

HTML é a linguagem primáriamente utilizada para estruturar documentos, embora nem sempre tenha exercido apenas essa função. Hoje em dia, também tem como papel descrever semântica dentro das páginas, facilitando a interação de mecanismos automatizados com websites modernos.

CSS é utilizado para introdução de estilos avançados e semi-interativos em páginas da web, sendo capaz de gerenciar as cores, posicionamento, e até mesmo animações de elemento de uma página da web.

JavaScript a principal linguagem de programação utilizada em webdev, seu papel o de adicionar funcionalidade avançada a websites, sendo capaz de gerenciar ambos back e front-end.
